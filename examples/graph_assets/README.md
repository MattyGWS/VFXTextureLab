# Graph Asset examples

- `rock_material_generator.vfxgraph` is a reusable graph asset with typed Shape Mask and Base Tint inputs; Material, Height and Surface Variation outputs; grouped public controls; and one universal Random Seed.
- `../graph_asset_rock_composition.vfxgraph` embeds two instances of the generator and blends their complete materials.

Drag `rock_material_generator.vfxgraph` from your file manager onto a VFX Texture Lab canvas, use **Add Graph Asset…**, or register this folder from Node Library.
