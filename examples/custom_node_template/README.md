# Custom node template

1. Copy this folder into a location added under **Library → Custom Node Libraries…**.
2. Change `id` in `node.toml` to a permanent unique reverse-domain identifier.
3. Save `kernel.wgsl`; VFX Texture Lab will hot-reload it.
4. Open **Library → Custom Node Diagnostics…** for compiler feedback.

See `docs/CUSTOM_NODES.md` for the complete manifest and WGSL ABI.
