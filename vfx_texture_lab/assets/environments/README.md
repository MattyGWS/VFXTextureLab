# Bundled environment maps

These compact environment-lighting maps are derived from the tonemapped preview panoramas of the following Poly Haven CC0 HDRIs:

- Studio Small 02 — Greg Zaal
- Cayley Interior — Poly Haven contributors
- Overcast Soil — Poly Haven contributors
- Sunset in the Chalk Quarry — Poly Haven contributors

The source assets are available from https://polyhaven.com/hdris under the Creative Commons CC0 1.0 dedication. The bundled maps are resized and inverse-tonemapped for a small application footprint; they are intended for interactive material preview rather than production lighting or redistribution as source HDRIs.
